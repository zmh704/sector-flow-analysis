# A股板块资金流向分析系统

一个基于浏览器的静态分析页面，用于可视化展示A股行业板块和概念板块的资金流向数据。

## 功能特点

✅ **双视图展示**：行业板块和概念板块分开显示
✅ **直观配色**：红色表示净流入（正值），绿色表示净流出（负值）
✅ **横向柱状图**：Y轴显示板块名称，X轴显示主力净额（亿元）
✅ **灵活配置**：可自定义显示数量，正负值分别排序
✅ **多文件支持**：自动读取项目目录下所有JSON数据文件
✅ **响应式设计**：适配各种屏幕尺寸
✅ **统计卡片**：显示关键指标和分析总结

## 快速开始

1. 将所有JSON数据文件放入项目目录（与index.html同级）
2. 可选：更新 `list.json` 文件，列出所有JSON文件名
3. 直接用浏览器打开 `index.html` 文件即可

## 使用说明

### 控制面板

- **行业板块显示数量**：设置正负各显示多少个行业板块（默认5）
- **概念板块显示数量**：设置正负各显示多少个概念板块（默认10）
- **刷新图表**：点击按钮或修改数量后点击刷新，更新图表显示

### 图表说明

- **排序规则**：
  - 正值：按主力净额从大到小排序
  - 负值：按主力净额从小到大排序（绝对值最大的在最下面）
- **颜色**：红色 = 净流入，绿色 = 净流出
- **交互**：鼠标悬停查看详细数据（净额、成交额、股票数量）

### 统计卡片

页面底部显示6个关键指标：
1. 行业板块净流入最多（板块名称 + 净额）
2. 行业板块净流出最多
3. 概念板块净流入最多
4. 概念板块净流出最多
5. 行业板块总数统计
6. 概念板块总数统计

## 多文件合并

如果项目目录下有多个JSON文件（如不同日期的数据），页面会自动合并：

1. 在 `list.json` 中添加文件名列表
2. 页面启动时按顺序加载所有文件
3. 自动合并 `行业板块资金流向` 和 `概念板块资金流向` 数组
4. 取第一个文件的 `分析总结` 和 `生成时间`（可扩展）

- 行业板块：10（正负各10 = 共20条）
- 概念板块：20（正负各20 = 共40条）

## 故障排除

**Q: 页面显示"请将JSON数据文件放入项目目录"**
A: 确保index.html能访问到JSON文件，使用本地服务器运行或直接双击html文件

**Q: 图表不显示**
A: 检查浏览器控制台是否有错误，确保Chart.js CDN可访问

**Q: 数据显示"NaN"**
A: JSON数据格式有误，检查数值字段是否为数字类型

**Q: 多文件不合并**
A: 确保list.json格式正确，每个文件名用双引号包裹，逗号分隔
